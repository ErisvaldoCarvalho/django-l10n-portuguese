.. _glossary:

========
Glossary
========

.. glossary::

    campo
        Um atributo de um :term:`modelo`; um dado campo usualmente mapea diretamente
        uma unica coluna do banco de dados.

        Veja :doc:`/topics/db/models`.

    view genérica
        Uma função do tipo :term:`view` de ordem maior que provê uma abraimplementação genérica/abstrata
        de um padrão ou idioma comum encontrado no desenvolvimento de utroas views.

        Veja:`/ref/generic-views`.

    model
        Modelos armazenam os dados da sua aplicação .

        Veja:`/topics/db/models`.

    MTV
        "Model-template-view"; um padrão de desenvolvimento de software, similar em estilo ao 'MVC', mas que é uma melhor descrição do jeito como o Django faz as coisas.

        Veja:`the FAQ entry <faq-mtv>`.

    MVC
        `Model-view-controller`__;um padrão de desenvolvimento de software. Django :ref:` veja MVC para mais detalhes <faq-mtv>`.

        __ http://en.wikipedia.org/wiki/Model-view-controller

    project
        Um pacote python -- ex. um diretório de código -- que contém todos os
        settings de uma instância do DJango. Este pode incluir a configuração do
        banco de dados, opções específicas do Django, e configurações específicas da aplicação.

    property
        Também conhecido como "atributos gerenciáveis", e uma característica
        de python desde a versão 2.2. Esta é uma maneira elegante de criar atributos
        que são acessíveis como atributos, mas que podem ser implementados através de chamadas de métodos.

        Vejac:`property`.

    queryset
        um objeto que representa um conjunto de linhas a ser lido do banco de dados.

        Veja:`/topics/db/queries`.

    slug
        Um label curto que contém somente letras, números,
        underscores ou hífens. São usitlizados geralmente em URL's,
        por exemplo, uma típica url de blog.

        .. parsed-literal::

            https://www.djangoproject.com/weblog/2008/apr/12/**spring**/

        o ultimo bit (``spring``) é o slug.

    template
        um pedaço de texto que funciona como um formatador para a representação de um dado.
        Um template ajuda a abstrair a apresentação de um dado a partir do próprio dado.

        Veja:`/topics/templates`.

    view
        Uma função responsável por retornar uma página.
