Guia de instalação rápida
=========================

Antes que você possa usar o Django, você vai precisar instala-lo. Nos temos um
:ref:`guia completo de instalação <topics-install>` que abrange todas as
possibilidades; este guia ira guiar você em uma simples e minima instalação que
funcionará enquanto você caminha através da introdução.

Instalando o Python
-------------------

Sendo um web framework Python, Django requer o  Python. Ele funciona com qualquer
versao 2.3 e superiores, mais nos recomendamos instalar Python 2.5 ou posterior.
Se fizer isso, você não precisará criar um banco de dados ainda: Python 2.5 ou
posterior inclui uma base de dados leve chamado SQLite_.

.. _sqlite: http://sqlite.org/

Baixe o Python em http://www.python.org. Se você estiver rodando Linux ou Mac OS X,
você provavelmente já tem ele instalado.

.. admonition:: Django com Jython

    Se você usa Jython_ (Uma implementação Python para a plataforma Java), você
    ira precisar seguir alguns passos adicionais. Veja :ref:`howto-jython` para
    detalhes.

.. _jython: http://www.jython.org/

você pode verificar que o Python esta instalado digitando ``python`` do seu
terminal; você deve ver algo como::

    Python 2.5.1 (r251:54863, Jan 17 2008, 19:35:17)
    [GCC 4.0.1 (Apple Inc. build 5465)] on darwin
    Type "help", "copyright", "credits" or "license" for more information.
    >>>

Criado um banco de dados
------------------------

Se você instalou Python 2.5 ou posterior, você pode pular este passo agora.

Se não, ou se você gostaria de trabalhar com um mecanismo de banco de dados
"grande" PostgreSQL, MySQL, ou Oracle, consulte a :ref:`informação sobre
instalação de banco  de dados <database-installation>`.

Remova qualquer versão antiga do Django
---------------------------------------

Se você está atualizando sua instalação do Django de versão uma anterior, você vai
precisar :ref:`desinstalar a versão antiga do Django antes de instalar a nova
versão <removing-old-versions-of-django>`.

Instale o Django
----------------

Você tem três opções fáceis para instalar Django:

    * Instale a versão do Django :ref:`provida pelo distribuidor do seu sistema
      operacional <misc-distributions>`. Esta é a opção mais rápida para aqueles
      que tem sistemas operacionais que distribuem o Django.

    * :ref:`Instale uma release oficial <installing-official-release>`. Esta
      é a melhor abordagem para usuários que querem um número de versão estável
      e não estão preocupados em rodar uma versão ligeiramente mais antiga do
      Django.

    * :ref:`Instalar a versao de desenvolvimento mais recente
      <installing-development-version>`. Esta é melhor para usuário que querem as
      mais recentes e melhores funcionalidades e não tem medo de executar códigos
      novos.
     
.. aviso::

    Se você fizer um dos dois primeiro passos, mantenha seus olhos longe da partes
    da documentação marcadas como ** novo na versão de desenvolvimento**. Esta
    frase marca características que estão disponíveis apenas na versão de
    desenvolvimento do Django; Se você tentar utilizar elas com um release oficial
    elas não irão funcionar.
