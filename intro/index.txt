.. seealso::

    Se você é novo em Python_, você pode querer começar obtendo uma idéia do
    que é a linguagem. Django é 100% Python, então se você tiver um mínimo de
    conforto com Python você provavelmente irá tirar mais proveito do Django.

    Se você é inteiramente novo em programação, você pode querer começar com esta
    `lista de recursos de Python para não programadores`_.
   
    Se você já conhece algumas outras linguagens e quer aprender Python
    rapidamente, nos recomendamos `Dive Into Python`_ (Disponível também na
    `versão dead-tree`). Se isso não é muito o seu estilo, Tem muitos outros
    `livros sobre Python`_.

    .. _livros sobre Python: http://wiki.python.org/moin/PythonBooks
