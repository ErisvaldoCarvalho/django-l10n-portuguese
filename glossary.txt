.. _glossary:

=========
Glossario
=========

.. glossary::

    field
        Um atributo em um :term:`model`; um determinado campo que usualmente é
        mapeado diretamente para uma única coluna em uma base dados.

        Veja :ref:`topics-db-models`.

    generic view
        Uma função :term:`view` de alta-ordem que abstrai idiomas comuns e
        padrões encontrados no desenvolvimento de um view.

        Veja :ref:`ref-generic-views`.

    model
        Um modelo armazena os dado de sua aplicação.

        Veja :ref:`topics-db-models`.

    MTV
        Veja :ref:`mtv`.

    MVC
        `Model-view-controller`__;  é um padrão de software. O Django
        :ref:`segue o padrão MVC até certo ponto <mtv>`.

        __ http://en.wikipedia.org/wiki/Model-view-controller

    project
        Um pacote Python -- ou seja um diretório de código -- que contém toda as
        configurações para uma instância do Django. Isto incluem configuração de
        banco de dados, opções especificas do Django e configurações especificas
        das aplicações.

    property
        Também conhecida como "atributos gerenciados", e também uma
        característica comum do Python desde a versão 2.2. Retirado da
        `documentação da property`__:

            Propriedades sao o jeito mais decente de implementar atributos cujo
            seu uso se assemelha-se com o acesso a atributos, mais cujo sua
            implementação usa chamadas a métodos. [...] Você pode fazer isto
            apenas sobrescrevendo ``__getattr__`` e ``__setattr__``; mas
            sobrescrevendo ``__setattr__`` deixara todas as atribuições de
            atributos mais lentas consideravelmente, e sobrescrever
            ``__getattr__`` eh sempre um pouco complicado de fazer corretamente.
            propriedades deixam voce fazer isto descomplicadamente, sem ter que
            sobrescrever ``__getattr__`` ou ``__setattr__``.

        __ http://www.python.org/download/releases/2.2/descrintro/#property

    queryset
        Um objeto representando algum conjunto de linhas à serem trazidas do
        banco de dados.

        Veja :ref:`topics-db-queries`.

    slug
        Uma pequena etiqueta para algo, contendo apenas letras, números,
        underscores ou hífens. São usados geralmente em URLs. Por exemplo, em uma
        URL típica de uma página em um blog.

        .. parsed-literal::

            http://www.djangoproject.com/weblog/2008/apr/12/**spring**/

        a última parte (``spring``) é um slug.

    template
        Um bloco de texto que separa a representação de um documento de seus dados.

        Veja :ref:`topics-templates`.

    view
        Uma função responsável por renderizar uma página.
